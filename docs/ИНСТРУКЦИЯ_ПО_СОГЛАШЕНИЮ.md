# Инструкция по пользовательскому соглашению

## Обновленное соглашение

Пользовательское соглашение было обновлено для комплексной системы управления PR-отделом BEST Москва и теперь включает:

1. **Управление задачами** - правила выполнения задач, дедлайны, ответственность
2. **Участие в событиях** - регистрация и участие в мероприятиях
3. **Управление оборудованием** - заявки, выдача, возврат (сохранено из предыдущей версии)
4. **Галерея** - правила размещения контента
5. **Геймификация** - система баллов и достижений
6. **Расширенная обработка персональных данных** - с учетом всех функций системы

## Файлы

- **MD файл**: `docs/BEST_PR_System_Agreement.md` - исходный текст соглашения
- **PDF файл**: `docs/BEST_PR_System_Agreement.pdf` - для печати и распространения (генерируется скриптом)

## Генерация PDF

Для создания PDF файла из MD файла используйте скрипт:

```bash
python scripts/create_agreement_pdf.py
```

Или установите зависимости и запустите:

```bash
pip install reportlab
python scripts/create_agreement_pdf.py
```

## Использование в системе

Соглашение автоматически загружается через API endpoint `/registration/agreement` и отображается:

1. **При регистрации** - пользователь должен принять соглашение перед регистрацией
2. **В Telegram боте** - через команду просмотра соглашения
3. **В веб-интерфейсе** - при необходимости просмотра

## Версионирование

Текущая версия соглашения: **2.0**

При изменении соглашения:
1. Обновите версию в `AgreementService.AGREEMENT_VERSION`
2. Обновите дату вступления в силу в MD файле
3. Перегенерируйте PDF файл
4. Уведомите пользователей об изменениях

## Структура соглашения

Соглашение включает 17 разделов:

1. Определения и термины
2. Предмет соглашения
3. Процедура использования системы
4. Выдача, использование и возврат оборудования
5. Ответственность пользователя
6. Ответственность администрации
7. Обработка персональных данных
8. Конфиденциальность
9. Ограничение и блокировка доступа
10. Ограничение ответственности администрации
11. Порядок разрешения споров
12. Подсудность
13. Обстоятельства непреодолимой силы
14. Срок действия соглашения
15. Изменение условий соглашения
16. Заключительные положения
17. Контактные данные администрации
