# 🎨 UI/UX Концепция BEST PR System

## 🎯 Принципы дизайна

### 1. Прогрессивное раскрытие
- **Новички** видят только необходимое
- **Опытные** получают расширенный функционал
- **Координаторы** имеют полный контроль

### 2. Интуитивность
- Понятная навигация
- Минимум кликов до цели
- Визуальные подсказки

### 3. Мотивация
- Геймификация на видном месте
- Прогресс всегда виден
- Положительная обратная связь

### 4. Производительность
- Быстрая загрузка
- Плавные анимации
- Оптимизированные запросы

## 🏠 Главная страница

### Для новичков (0-100 баллов)

**Верхняя секция:**
```
┌─────────────────────────────────────┐
│  👋 Добро пожаловать в BEST PR!    │
│                                     │
│  Ты только начинаешь? Отлично!      │
│  Выбери свою первую задачу 👇      │
└─────────────────────────────────────┘
```

**Горячие задачи:**
- Карточки с простым описанием
- Большая кнопка "Попробовать себя"
- Иконки типов задач (SMM, Design, Channel)
- Счётчик дней до дедлайна

**Быстрые действия:**
- 📝 Подать заявку на задачу
- 📚 Пройти онбординг
- 👥 Написать координатору

**Ограничения:**
- Не видит все задачи (только открытые)
- Не видит календарь (только свои задачи)
- Не видит рейтинг (только свой прогресс)

---

### Для участников (101-500 баллов)

**Дашборд:**
```
┌─────────────────────────────────────┐
│  📊 Моя статистика                  │
│  • Активных задач: 3                │
│  • Баллов: 250                       │
│  • Уровень: Активный 🔵             │
└─────────────────────────────────────┘
```

**Секции:**
1. **Мои задачи** - активные задачи с прогрессом
2. **Ближайшие дедлайны** - календарь на неделю
3. **Доступные задачи** - список с фильтрами
4. **Мой прогресс** - график баллов, уровень

**Дополнительно:**
- Видит все задачи
- Может фильтровать
- Видит календарь
- Видит свой рейтинг

---

### Для активных участников (501-1500 баллов)

**Расширенный дашборд:**
- Полная статистика
- Графики активности
- История выполненных задач
- Сравнение с другими

**Дополнительно:**
- Полный таймлайн

- ТОП-10 рейтинг

## 📋 Страница задач

### Список задач

**Фильтры (слева):**
- Тип задачи (SMM, Design, Channel, PR-FR)
- Статус (Открыта, В работе, На проверке)
- Приоритет (Низкий, Средний, Высокий, Критичный)
- Мероприятие (BIP, TW, BESTALKS...)
- Срок выполнения

**Карточка задачи:**
```
┌─────────────────────────────────────┐
│  🎬 Промо-ролик для BIP             │
│  Channel • Приоритет: Высокий 🔴    │
│                                     │
│  📅 Дедлайн: 20 февраля            │
│  👥 Нужны: видеограф, монтажёр      │
│  🎯 Баллов: 300                     │
│                                     │
│  [📝 Подробнее] [✅ Взять задачу]   │
└─────────────────────────────────────┘
```

**Цветовая индикация:**
- Зелёная рамка - в работе
- Жёлтая рамка - согласование
- Красная рамка - срочно/дедлайн
- Серая рамка - выполнено

## 🗓️ Календарь и таймлайн

### Календарный вид

**Месячный календарь:**
- Мероприятия выделены цветом
- Задачи отображаются точками
- Клик по дню → список задач

### Таймлайн вид

**Горизонтальная шкала:**
```
Февраль          Март            Апрель
│────────────────│────────────────│
│ 🟢 BIP промо   │ 🟡 TW съёмка   │
│ 🟢 BIP дизайн  │ 🔴 BESTALKS    │
│                │ 🟣 ЯВ          │
```

**Цветовая система:**
- 🟢 Зелёный - этап в работе
- 🟡 Жёлтый - согласование
- 🔴 Красный - дедлайн
- 🟣 Фиолетовый - мероприятие
- 🔵 Синий - выполнено

**Интерактивность:**
- Наведение → детали задачи
- Клик → открыть задачу
- Перетаскивание → изменить дату (для координаторов)

## 🎬 Channel Production (специальная вкладка)

### Мастер создания задачи

**Шаг 1: Основная информация**
- Название
- Описание/ТЗ
- Мероприятие
- Приоритет

**Шаг 2: Этапы производства**
- Выбор этапов (чекбоксы)
- Установка дат для каждого этапа
- Выбор: готовый сценарий или разработка

**Шаг 3: Оборудование**
- Автоматическое предложение
- Выбор нужного оборудования
- Даты бронирования

**Шаг 4: Участники**
- Выбор нужных ролей
- Автоматическое создание чата

- Изменение статуса заявки
