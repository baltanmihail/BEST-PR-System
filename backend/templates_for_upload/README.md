# Инструкция по загрузке шаблонов задач в Google Drive

## Структура папок

Шаблоны должны быть загружены в следующие папки:

1. **Пост в соцсети.json** → `Admin/Coordinators/SMM/`
2. **Графика для поста.json** → `Admin/Coordinators/Design/`
3. **Видеоролик.json** → `Admin/Coordinators/Channel/`
4. **Организация мероприятия.json** → `Admin/Coordinators/PR-FR/`

## Как загрузить

1. Откройте Google Drive
2. Перейдите в папку `BEST PR System/Admin/Coordinators/`
3. Откройте нужную подпапку (SMM, Design, Channel или PR-FR)
4. Загрузите соответствующий JSON файл в эту папку
5. Повторите для всех файлов

## Важно

- Файлы должны иметь точные имена, как указано выше
- После загрузки система автоматически обновит ID в файлах (когда шаблоны будут созданы в базе данных)
- Если файл уже существует, система обновит его содержимое

## Примечание

Если шаблоны уже созданы в базе данных, но файлы не загружены на Drive - это не критично. Система будет работать, но файлы на Drive полезны для:
- Резервного копирования
- Просмотра/редактирования вручную
- Двусторонней синхронизации
